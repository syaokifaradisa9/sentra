
Branches  
- BranchFlowsRegressionTest.php – Regression of branch flows (list, create, edit, delete, export).  
- ManageBranchTest.php – Ensures branch CRUD works for Businessman users.  
- BranchServiceIntegrationTest.php – Verifies the branch service communicates correctly with repositories/DB.  

Business  
- BusinessFlowsRegressionTest.php – Regression for business pages.  
- ManageBusinessTest.php – Business CRUD scenarios driven by Businessman role.  
- BusinessServiceIntegrationTest.php – Integration coverage for the business service layer.  

Categories  
- CategoryFlowsRegressionTest.php – Regression for listing/printing categories.  
- ManageCategoryTest.php – Validates full CRUD operations on categories.  
- CategoryServiceIntegrationTest.php – Integration between category service and repositories.  

Products  
- ProductFlowsRegressionTest.php – Regression for product UI flows.  
- ManageProductTest.php – Product CRUD assertions.  
- ProductServiceIntegrationTest.php – Integration coverage for product service.  

Promos  
- tests/Feature/Promo/* – Create, edit, delete, and export promo scenarios.  

Middleware & Roles  
- EnsureBranchOwnerTest.php (Feature) – Only branch owners pass this middleware.  
- EnsureBusinessOwnerTest.php (Feature & Unit) – Middleware validation for business ownership.  
- RoleFeatureAccessTest.php – Role matrix assertions by HTTP status:  
  - Businessman → all modules (dashboard, cashier, business, branches, categories, products, promos) return 200.  
  - BusinessOwner → all modules except business.index (expected 403).  
  - SmallBusinessOwner & Manager → only dashboard/cashier/categories/products/promos return 200; business & branches must be 403.  
  - Cashier → only dashboard/cashier/categories/products/promos return 200; business & branches return 403.  

DTO, Repository, Service  
- BranchDTOTest.php, BusinessDTOTest.php, CategoryDTOTest.php, ProductDTOTest.php – DTO construction/serialization checks.  
- Repository tests (Branch, Business, Category, CategoryBranch, Product, ProductBranch) – Expected behavior of each Eloquent repository.  
- Service tests (BranchService, BusinessService, CategoryService, ProductService) – Verifies domain logic inside service layers.
